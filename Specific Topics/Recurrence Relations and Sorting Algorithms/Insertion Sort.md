Sort the input array by inserting the element we are on into its correct location.

Loop Invariant:

A weaker version of [[Selection Sort]] where:

arr\[1...i-1] is sorted.

Time Complexity:
* Best: O(n)
* Worst: O(n^2)
* Average: O(n^2)
Space Complexity: O(N)
Auxiliary Complexity: O(1)

[[Stability|Stable]]: Yes

In-Place: Yes
